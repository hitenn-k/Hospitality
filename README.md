#Hospitality Dashboard 🏨📊

#Overview#
This project is a Power BI dashboard designed to analyze and visualize key metrics in the hospitality industry. It provides insights into revenue, occupancy, ratings and total guests, helping managers make data‑driven decisions.


Objectives

  •	Track occupancy rates and seasonal trends.
  •	Monitor revenue performance across time periods.
  •	Analyze customer demographics and preferences.
  •	Provide actionable insights for improving service efficiency.


Key Insights & Visuals

  📈 Booking % by Platform – Compare online vs offline bookings, OTAs vs direct channels.
  👥 Total Guests by Category – Segment guests into business, leisure, family, and group travelers.
  🌆 City‑wise Revenue & Occupancy % – Track performance across different locations.
  ⭐ Average Rating – Monitor guest satisfaction scores.
  📅 Week‑wise Trends (Field Parameter) – Dynamic analysis of bookings, revenue, or occupancy by week.
  🏨 Occupancy % by Day Type – Compare weekdays vs weekends.
  💰 Occupancy % by Revenue Type (Button Toggle) – Switch between room revenue, F&B revenue, or combined metrics.


Tools & Techniques

  •	Power BI Desktop – Data modeling, DAX measures, interactive visuals.
  •	Power Query – Data cleaning and transformation.
  •	DAX – Custom measures for occupancy %, revenue growth, and YoY comparisons.


Project Goals

  •	Demonstrate practical data analysis skills using Power BI.
  •	Showcase ability to build interactive dashboards for business decision‑making.
  •	Serve as a portfolio project for data analyst roles.
